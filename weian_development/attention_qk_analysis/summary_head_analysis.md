# 注意力频段分析总结

## 背景与目标
- 希望用尽量少、直观的统计量来描述一个注意力头的整体行为，而不是逐位置精确还原。
- 幅值部分 `|Q| * |K|` 已经易于理解；难点在于相位（`cos(θ_q - θ_k)`），以前直接平均相位会让重构变差。
- 目标：找到一套数学上严谨、实现简单的指标，能够映射到 `direct q·k` 曲线的大致趋势，帮助理解模型在不同频段和相对距离上的注意力模式。

## 核心问题
1. **相位与幅值耦合**
   - 简单平均 `cos δ` 隐含“幅值与相位差独立”的假设。
   - 实际上，近邻配对幅值大、相位集中；远距配对幅值小、相位像噪声，简单平均会被远距噪声拉偏。
2. **RoPE 的作用**
   - RoPE 为每个频段注入 `ω_f Δ` 的基波相位，同时模型还能学到额外偏移。
   - 统计时若不先逆 RoPE，就把基波和模型偏移混在一起，不利于分辨模型的真实行为。
3. **Yarn 缩放的影响**
   - DeepSeek-R1-Qwen3 使用 Yarn RoPE，cos/sin 在生成时会乘以全局 `attention_scaling`。
   - 捕获的 Q/K 都带有该缩放；若逆 RoPE 时忽略它，会让“还原”的向量系统性放大，导致平均夹角、复相关等统计量全部失真。
   - 最新脚本在逆 RoPE 前先除以该缩放，验证闭环误差小于 1e-6。

## 推荐统计方案
1. **逆 RoPE，再建模**
   - 捕获的 Q/K 先逆 RoPE，得到复数写法 `q_tilde`, `k_tilde`。
2. **计算复相关 (频段、可选分桶)**
   - 基本量：`C_f = E[q_tilde * conjugate(k_tilde)] = A_f + i B_f`。
   - 若需要区分距离，先算 `C_f(Δ)`，再按权重 `p(Δ)` 聚合到 `C_f`。
   - 权重可取配对频率、真实注意力，或关注窗口（如 `Δ ≤ 32`）。
3. **从 `C_f` 读出指标**
   - `abs(C_f)`：频段贡献强度（幅值 × 相位综合效果）。
   - `arg(C_f)`：主导方向，近似“平均相位差”。
   - `reconstruction(Δ) = Σ_f [Re(C_f) cos(ω_f Δ) − Im(C_f) sin(ω_f Δ)]`：可以直接与 `direct q·k` 曲线比较。实现中同时绘制真实的预 softmax `Q·K` 聚合曲线，用于确认统计量是否贴合 Ground Truth。
4. **可选地对 `C_f(Δ)` 做分桶**
   - 例如 “近邻 (Δ ≤ 32)” vs “远距 (Δ ≥ 256)” 两组 `C_f`，只需四个数字就能描述一个 head 行为。
   - 再结合 `abs(C_f) / E[|Q||K|]` 等简单比值，可反映相位一致性。

## 实验对比结论
- 只使用 RoPE 后的相位（未逆旋）会把位置基波混在统计里，重构更差；逆 RoPE 后再计算复相关效果更好。
- 旧方案（`E[|Q||K|]` + 平均 `φ_f`）在统计上不成立，尤其在远距噪声占多数时，重构会把曲线峰值推到右边。
- 使用复相关 `C_f` 重构时，误差能显著下降（Spearman 相关也翻倍），说明“幅值 × 相位”联合统计是正确路径。

## 实施步骤（供接手者参考）
1. **数据准备**：从 `qk.pt` 读取 Q/K，逆 RoPE 得到 `q_tilde`, `k_tilde`。
2. **统计复相关**：
   ```python
   corr = average(q_tilde * conjugate(k_tilde))      # 全局
   corr_delta = average(... per delta bucket ...)    # 如需区分距离
   ```
3. **指标输出**：
   - `abs(corr)`, `arg(corr)`
   - 若分桶：`abs(corr_near)`, `arg(corr_near)` 等
   - 可附带 `abs(corr) / mean(|Q||K|)` 表示相位一致性
4. **可视化**：
   - 幅值/相位条形图 + `direct q·k` 与重构曲线对比
   - 按 Δ 的 `corr_delta` 热力图以辅助理解
5. **评估**：使用 Spearman 等相关系数衡量统计指标对 `direct q·k` 的解释力。

## 建议下一步
- 针对具体项目需求，确定一个或两个权重函数 `p(Δ)` 来生成 “近邻” 与 “远距” 的复相关；
- 制作简洁的指标表（如 `|C_f^{near}|`, `arg(C_f^{near})`, `|C_f^{far}|`, `arg(C_f^{far})`）；
- 按频段排序或聚合，可快速识别每个 head 的关注模式；
- 根据这些指标挑选代表性 head，再做更细的 Δ 分桶或热力图分析。
