# Phase-Based Reconstruction: Mathematical Formulation

## 目标
我们希望用简单的频段统计量描述每个注意力头的行为，尤其是解释以下量之间的关系：

- `|Q|`, `|K|`：RoPE 还原后在每个频段的幅值（通过 `sqrt(q_x^2 + q_y^2)`）。
- `|Q||K|` 在自回归约束 `(i ≥ j)` 下的平均值（全局或随距离 Δ 的统计）。
- 平均相位差 `φ_f`（或对应的 `cos`, `sin` 分量）。
- 基于这些统计的重构公式，如 `Σ_f |Q||K| cos(ω_f Δ + φ_f)`。

下面从数学角度推导这些量与真实注意力点积 `Q_i·K_j` 之间的联系。

## 二维频段表示与 RoPE
对每个频段 `f`，RoPE 让 query/key 的偶数、奇数维形成一对二维向量。记：

- `q_{i,f} = (q_x, q_y)`，`k_{j,f} = (k_x, k_y)`，均为 RoPE 后的向量。
- 对应的复数记法：`\tilde{q}_{i,f} = q_x + i q_y`，`\tilde{k}_{j,f} = k_x + i k_y`。
- 幅值：`|Q_{i,f}| = |\tilde{q}_{i,f}|`，`|K_{j,f}| = |\tilde{k}_{j,f}|`。
- 相位：`θ_{q,i,f} = \arg \tilde{q}_{i,f}`，`θ_{k,j,f} = \arg \tilde{k}_{j,f}`。
- 叉积与点积：
  - `q_{i,f}·k_{j,f} = |Q_{i,f}| |K_{j,f}| cos(θ_{q,i,f} - θ_{k,j,f})`
  - `\det[q_{i,f}, k_{j,f}] = |Q_{i,f}| |K_{j,f}| sin(θ_{q,i,f} - θ_{k,j,f})`

在 RoPE 之前的复数表示 `q^{orig}`、`k^{orig}` 与 RoPE 后的 `q`, `k` 存在线性旋转关系（一个相位因子）。倒推 RoPE 时，我们用 cos/sin 表格把旋转去除，得到 `q^{orig}`, `k^{orig}`，再拆成二维向量/复数可以直接得到上述幅值与相位。

## 单个 (i, j) 对的注意力贡献
对固定的 query 位置 `i` 和历史 key 位置 `j`，RoPE 后的点积为：

```
Q_i · K_j = Σ_f q_{i,f} · k_{j,f}
          = Σ_f |Q_{i,f}| |K_{j,f}| cos(θ_{q,i,f} - θ_{k,j,f})
```

若我们只关心“在距离 Δ = i - j 下的平均行为”，可以对所有满足该 Δ 的 `(i, j)` 做平均：

```
E_Δ[Q_i · K_j] = Σ_f E_Δ[ |Q_{i,f}| |K_{j,f}| cos δ_{ij,f} ]
```
其中 `δ_{ij,f} = θ_{q,i,f} - θ_{k,j,f}`。

这给出一个自然的“傅立叶分量”：每个频段贡献 `E_Δ[ |Q||K| cos δ ]`。

## 全局平均与分量分解
如果我们不区分 Δ，而是对所有 `(i ≥ j)` 配对求平均，得到：

```
E[Q_i · K_j] = Σ_f E[ |Q||K| cos δ ]
```
我们用以下符号来记这些分量：

- `A_f = E[ |Q||K| cos δ ]`
- `B_f = E[ |Q||K| sin δ ]`
- `W_f = E[ |Q||K| ]`
- `φ_f = atan2(E[sin δ], E[cos δ])`

理论上，原始点积的平均可写成：

```
E[Q_i · K_j] = Σ_f A_f
```
而如果我们想把“平均幅值”与“平均相位”结合成 `R_f cos(ω_f Δ + φ_f)` 形式，可以利用三角恒等式：

```
A_f cos(ωΔ) - B_f sin(ωΔ) = R_f cos(ωΔ + φ_f)
```
其中 `R_f = sqrt(A_f^2 + B_f^2)`，`φ_f = atan2(B_f, A_f)`。

这就是把 `cos`、`sin` 两个正交基的系数换成幅度+相位的另一种表示。

## 现有实现的近似
目前脚本为了简单，使用了：

```
E[|Q||K| cos δ] ≈ E[|Q||K|] · E[cos δ]
E[|Q||K| sin δ] ≈ E[|Q||K|] · E[sin δ]
```
并将 `φ_f = atan2(E[sin δ], E[cos δ])` 带入重构
```
Σ_f E[|Q||K|] cos(ω_f Δ + φ_f)
```
这一步隐含假设：`|Q||K|` 与 `cos δ`、`sin δ` 独立。而实际数据中，近邻配对往往幅值更大、相位更对齐，远距配对幅值小且相位噪声大。如果直接全局平均 `E[cos δ]`、`E[sin δ]`，远距噪声会把平均角推向 ±π/2，同时 `E[|Q||K|]` 仍然很大，于是近邻的贡献反而被削弱。

从数学角度看，这是因为我们忽略了 `cos δ` 与幅值之间的相关性。更准确的做法是直接使用 `A_f`、`B_f`（或在统计时按 Δ 分桶），而不是用“平均幅值 × 平均相位”来近似。

## 为什么翻转 φ_f 不能根治
将 `cos(ωΔ + φ_f)` 改成 `cos(ωΔ - φ_f)` 实际上就是把 `B_f` 取负。若全局 `B_f` 本来就被远距配对主导，取负只能略微修正平均方向，但仍无法恢复“近邻幅值大”的事实。这解释了为什么把 `φ_f` 取负虽然略降 MSE，峰值依旧在右侧。

## 下一步可能的改进
数学上，更合适的统计方法是直接把二维频段看成复数，并求期望：

```
C_f(Δ) = E\big[q̃_{i,f} · \overline{k̃_{j,f}} \mid i-j = Δ\big]
```

其中 `q̃ = q_x + i q_y`，`k̃ = k_x + i k_y` 是 RoPE 还原后的复数表示。该期望自然包含幅值与相位：

```
C_f(Δ) = A_f(Δ) + i·B_f(Δ)
```

若要得到单一指标，可对 `Δ` 再加权平均：

```
C_f = Σ_Δ p(Δ) · C_f(Δ)
```

其中 `p(Δ)` 可选配对频率、真实注意力权重，或用户关心的窗口。最终：

```
R_f = |C_f|,
φ_f = arg(C_f).
```

这种复相关系数统计无需假设幅值与相位独立，且天生记录“谁的贡献更强”。随后若需要画 `Σ_f R_f cos(ω_f Δ + φ_f)` 之类的曲线，直接把 `A_f = Re(C_f)`、`B_f = Im(C_f)` 带回去即可。

## 小结
- `Q_i · K_j` 可以看作频段的傅立叶和，其系数是 `|Q||K| cos δ`（余弦）和 `|Q||K| sin δ`（正弦）。
- 将频段平均转为 `R_f cos(ωΔ + φ_f)` 是严格的三角恒等式，但前提是使用 `A_f`, `B_f` 作为系数。
- 简单地用 `E[|Q||K|]` 与 `E[cos δ], E[sin δ]` 相乘忽略了幅值-相位的依赖关系，导致统计被远距配对主导，进而在重构中压低近邻贡献。
- 若要用简单统计量描述 head 行为，需要先决定“想反映的是全局平均还是某个距离范围”，然后按这个范围收集 `A_f/B_f` 或 `φ_f`，再进行重构。
