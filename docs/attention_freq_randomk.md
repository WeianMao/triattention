# 随机 Key 周期重建公式摘要

本文记录 `freq_magnitude_single_plot_meanvec_randomk.py` 中使用的核心频域重建公式，便于后续分析和复现。

## 基础统计

- 序列长度记为 `T`，频率维度记为 `F`。
- 逆 RoPE 后得到的复数表示：
  - 查询：`q_{t,f} ∈ ℂ`
  - 键：`k_{t,f} ∈ ℂ`

平均向量和幅值：

```
E[q_f] = (1 / T) * Σ_{t=1}^T q_{t,f}
E[k_f] = (1 / T) * Σ_{t=1}^T k_{t,f}
|E[q_f]| = abs(E[q_f])
|E[k_f]| = abs(E[k_f])
```

相位差估计：

```
φ̂_f = arg(E[q_f] * conj(E[k_f]))
```

## 平均向量重建

脚本原始的重建项：

```
S_mean(Δ) = Σ_{f=1}^F |E[q_f]| * |E[k_f]| * cos(ω_f * Δ + φ̂_f)
```

其中 `ω_f` 为 RoPE 对应的角频率，`Δ` 为 token 间距。

## 单个 key 替换重建

新增的三幅图每次随机抽取一个 token 索引 `τ`，用其键向量代替 `E[k_f]`：

```
φ_f(τ) = arg(E[q_f] * conj(k_{τ,f}))
S_random(Δ; τ) = Σ_{f=1}^F |E[q_f]| * |k_{τ,f}| * cos(ω_f * Δ + φ_f(τ))
```

脚本会使用固定随机种子采样最多三个 `τ`，并绘制对应的 `S_random(Δ; τ)` 曲线，以对比平均向量与单个实例的差异。
