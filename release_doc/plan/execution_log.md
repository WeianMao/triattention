# 执行日志

> **所有 agent（Controller 和 Executor）必须维护此日志。** 中断后下一个 agent 靠此文件恢复断点。

## 日志格式

每完成一个步骤，按以下格式追加记录：

```markdown
## Step X.Y: [步骤名]
- **Agent**: controller / executor
- **时间**: YYYY-MM-DD
- **状态**: 完成 / 部分完成 / 失败
- **执行内容**: [具体做了什么]
- **已知问题**: [如有]
- **不符合目标之处**: [如有，附原因]
- **下一步**: [建议]
```
